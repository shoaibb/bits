With this tip you can download courses from Coursera . . .

Recently, I found some interesting courses that I wanted to study and I got enrolled in these. However, due to a busy schedule and lack of free time, I was not able to complete the courses in time. Then I thought how wonderful it will be if I can bulk download these courses and can study whenever I get free time.

I found a nice tool dl_coursera from this repo: https://github.com/FLZ101/dl_coursera
The good thing with this tool is that you can download courses as well as specializations through it. However, there are certain requirements to fulfil when using dl_coursera. First, we can install dl_coursera using pip as follows:

1

pip install dl_coursera

Steps to download courses:

  1. Next, install a google-chrome plugin know as Get cookies.txt LOCALLY from Google chrome store https://chrome.google.com/webstore/detail/get-cookiestxt-locally/cclelndahbckbenkjhflpdbgdldlbecc
  2. Login to coursera and click on the Get cookies.txt LOCALLY and export cookies of the logged-in courersa page to a file locally, lets call it cookies.txt
  3. Now, get the slug for a course or specialization from coursera that you want to download. A slug is a shortname for the course/specialization and can be found usually at the very last of the url, e.g. in this url https://www.coursera.org/learn/intermediate-postgresql the slug will be intermediate-postgresql
  4. Now use this command to download the course:
1

dl_coursera --cookies path/to/cookies.txt --slug intermediate-postgresql --outdir dir-to-store --how builtin

We can provide the flag --isSpec to download a complete specialization instead of a single course.

1

dl_coursera --cookies path/to/cookies.txt --slug intermediate-postgresql --isSpec --outdir dir-to-store --how builtin
