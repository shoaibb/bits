Here are some Linux tips and tricks (basics to advanced level).

Shortcut/symlink to a storage location

The following command will create a shortcut/symlink to the directory **/run/user/310617548/gvfs/smb-share:server=10.1.111.10,share=forskning03$/Dioxinlab/mzML-files/ at the current location. The shortcut name will be files_to_move.

1

sudo ln -s '/run/user/310617548/gvfs/smb-share:server=10.1.111.10,share=forskning03$/Dioxinlab/_mzML-files/' files_to_move

Bulk files creation

The following command will create 500 small files each 1MB in size:

1

for i in {001..500}; do head -c 1M </dev/urandom >smaller_files/randfile$i; done

Search files

Use the following command to find files using partial match in the file name

1

find . -type f -name "*John*"
